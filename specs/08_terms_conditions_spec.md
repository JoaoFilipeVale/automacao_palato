# Terms and Conditions Specification

## 1. Overview
This specification defines the testing requirements for the "Termos e Condições de Uso" page.
Test implementation: `tests/test_legal_pages.py`

- **URL**: `/termos-e-condicoes-de-uso/`
- **Title**: Termos e Condições de Uso - Palato Digital

## 2. Key Sections to Verify
The page content is structure by numbered headers. The test should verify the visibility of:
1.  **Main Title**: "Termos e Condições do Palato Digital"
2.  **Section 1**: Aceitação dos Termos
3.  **Section 2**: Direitos de Propriedade Intelectual
4.  **Section 3**: Uso Correto do Website
5.  **Section 4**: Limitação de Responsabilidade
6.  **Section 5**: Ligações para Websites de Terceiros
7.  **Section 6**: Lei Aplicável e Foro
8.  **Section 7**: Alterações a estes Termos
9.  **Section 8**: Contacto

## 3. Layout Verification (Standard)
- **Header**: Logo and Menu visible.
- **Footer**: Footer visible.
